# Privy — Private Chat

A secure, end-to-end encrypted one-on-one chat app with real-time messaging, typing indicators, dark/light mode, and PWA support.

## ⚠️ Demo Only

This is a **demo project**. The Firebase backend is disabled for security.

To run this app:
1. Create your own Firebase project
2. Enable Firestore
3. Add your config to `firebase-config.js`

## Features

- 🔐 End-to-end encryption
- 💬 Real-time messaging
- ⌨️ Live typing indicators
- 🌓 Dark/Light mode
- 📱 PWA installable
- 🔗 6-digit code pairing

## Tech Stack

- HTML5 / CSS3 / JavaScript
- Firebase Firestore
- Web Push API
